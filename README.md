# Mood Tracker

## Overview

Mood Tracker is a simple web application built using HTML, CSS, JavaScript, and Tailwind CSS. Users can select and track their mood through an interactive and responsive interface.

This was my first project after learning front-end web development fundamentals and helped me strengthen my understanding of DOM manipulation, event handling, responsive design, and UI development.

## Features

* Select and track daily mood
* Interactive user interface
* Responsive design using Tailwind CSS
* Dynamic updates using JavaScript
* Clean and beginner-friendly code structure

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Tailwind CSS
